This includes software such as Slack which is generally an instant messaging and collaboration system. You can have a Slack workspace were you share with other members to communicate and get work done. All content inside Slack is searchable, including files, conversations, and users. Slack was launched in August 2013. More types of technology are Git and Github. Git is a version control system for monitoring changes in computer files and coordinating work on those files among other people. It is mainly used for code management in software development. Github is an online software used to store files from Git itself.
One risk is sharing files via Slack or Email. You could recieve a document as an attachment into a group chat instead of a private chat. Anyone within the chat would see this document and can Copy without permission. Email on the other hand is not designed to be secure. Anyone with access to an intermediate mail server can easily retrive the documents, which may contain private information or design templates.
